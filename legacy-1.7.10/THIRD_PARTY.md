# Included build dependencies

The `deps` directory contains compile-time tools and libraries so this archive
can be built offline. None of these dependency JARs are copied into the final
RenderScale JAR.

- Minecraft Forge 1.7.10-10.13.4.1614 universal — LGPL-2.1
- LaunchWrapper 1.12 — Mojang/Minecraft library
- ASM All 5.0.3 — BSD-3-Clause
- LWJGL 2.9.1 — BSD-3-Clause
- Eclipse Compiler for Java 3.33.0 — Eclipse Public License 2.0

The final mod remains subject to the license in `LICENSE`.
